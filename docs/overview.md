## 📖 Struktura projektu

**Hybridní architektura:** Symfony backend + Twig templating + React micro-apps  
**Databáze:** PostgreSQL (app data) + MSSQL (INSYZ data)  
**Frontend:** Tailwind CSS + BEM + Material React Table  
**Development:** DDEV + Mock INSYZ data
